## Description

API built with [Nest](https://github.com/nestjs/nest) and [Prisma](https://prisma.io).

## Installation

```bash
$ pnpm install
$ docker compose up -d
$ npx prisma migrate dev
```

## Running the app

```bash
# development
$ pnpm run start

# watch mode
$ pnpm run start:dev

# production mode
$ pnpm run start:prod
```

## Test

```bash
# unit tests
$ pnpm run test

# e2e tests
$ pnpm run test:e2e

# test coverage
$ pnpm run test:cov
```
